AMCA H5 · 图片目录使用说明
============================

把图片按下列文件名放进这个 images/ 文件夹即可：

  cover.jpg                 ← 第一屏（封面）整页背景图（推荐 1080×1920 竖屏）
  img-01.jpg ~ img-26.jpg   ← 内文 26 个图片占位框（详见图片清单）

支持的格式：jpg / jpeg / png / webp / avif
（脚本会自动按顺序尝试，最优先 jpg）

文件命名一定要严格匹配：
  ✗ Cover.JPG / IMG-1.PNG / img_01.jpg / 封面.jpg
  ✓ cover.jpg / img-01.jpg

详见根目录 《图片清单_26张.md》
